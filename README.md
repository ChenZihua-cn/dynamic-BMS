# 动态物理先验的贝叶斯符号回归

## 选题背景

封闭形式的数学模型是人类理解世界的重要工具，符号回归是一类自动搜索解析表达式的机器学习方法，其目标是从数据中直接发现封闭形式的数学表达式。

现有的方法大致有:
- [遗传编程方法]()
- [稀疏回归方法]()
- [深度学习方法]()

现有Bayes Machine Scientist (BMS) 的局限性:
- 传统方法的选择策略均是启发性的，复杂度惩罚和适应度函数均无统一理论框架，是手动设置的
- 搜索表达式是一个NP-hard问题，无法保证方法可以系统地探索模型空间
- 先验知识的使用是静态的，无论是流体力学还是天体物理，机器都用同样的先验来评价所有候选模型

本课题的核心思想是：

**将人类物理学家的多层次物理直觉——量纲分析、极限行为、对称性、参数物理范围等——形式化为多个“专家先验”，并通过一个动态门控网络根据数据特征和当前模型结构自适应地选择或组合这些先验。**

这一想法源于对BMS回归框架的理解和对混合专家模型（Mixture of Experts, MoE）范式的借鉴：
- 贝叶斯符号回归提供了一个将先验知识系统性地纳入模型评估的概率框架
- MoE架构提供了“分而治之”的动态路由机制，使模型能够根据输入特征自适应地选择最合适的专家

将两者结合，可以构建一个能够根据问题领域和数据特性动态调整其“科学品味” 的机器科学家。

## 研究现状


## 研究目标

建立 “动态物理先验的贝叶斯符号回归”（Dynamic Physics Priors for Bayesian Symbolic Regression, DP-BSR） 的理论框架与实现方法，使机器科学家能够根据问题领域和数据特征自适应地调整其先验期望，从而更高效、更准确地从数据中发现物理自洽的数学模型。


## 研究内容

**研究内容一：专家先验的形式化与构建**

设计并实现一组可插拔的“专家先验”，每个专家先验编码一种特定的物理直觉或归纳偏置：

| 专家先验 | 物理直觉 | 形式化方式 |
|---------|---------|-----------|
| **量纲分析专家** | 物理方程必须量纲一致 | 硬约束：在树生成/替换时强制检查量纲兼容性；软约束：惩罚量纲不一致的表达式 |
| **极限行为专家** | 模型在特定极限下（如 \( x \to 0 \), \( x \to \infty \)）应表现正确 | 软先验：根据模型极限值与理论预期值的偏差调整先验概率 |
| **对称性专家** | 物理系统往往具有对称性 | 软先验：偏好具有特定对称性（如交换对称、平移对称）的表达式 |
| **参数范围专家** | 物理参数（如质量、耦合常数）必须在物理允许范围内 | 软先验：参数落在非物理区域时降低先验概率 |
| **复杂度专家（奥卡姆剃刀）** | 更简单的表达式更可取 | 软先验：偏好树结构更简洁的表达式 |
| **渐近匹配专家** | 模型应匹配已知的微扰/极限理论 | 软先验：根据与已知渐近展开的偏差调整概率 |

**研究内容二：动态门控网络的设计**

门控网络是DP-BSR的核心决策模块，负责根据输入动态计算各专家先验的权重。

- **输入特征**：数据特征（样本量、噪声水平、变量维度、数值范围等）和当前候选表达式的结构特征（树深度、算子类型分布等）。
- **架构设计**：采用轻量级神经网络或注意力机制，输出一个概率分布 \( \mathbf{g} = [g_1, g_2, \ldots, g_K] \)，其中 \( \sum_k g_k = 1 \)。
- **门控策略**：支持**软门控**（所有专家先验加权组合）和**硬门控**（仅激活Top-K专家）两种模式。

最终的组合先验为：

\[
p_{\text{total}}(f) = \prod_{k=1}^{K} p_{\text{expert}, k}(f)^{g_k(\text{data}, f)}
\]

或加性形式：

\[
\log p_{\text{total}}(f) = \sum_{k=1}^{K} g_k(\text{data}, f) \cdot \log p_{\text{expert}, k}(f)
\]

**研究内容三：动态先验与贝叶斯推理的融合**

将动态组合先验 \( p_{\text{total}}(f) \) 代入BMS的贝叶斯框架：

\[
p(f_i|D) \propto \exp\left[-\frac{B(f_i)}{2} + \log p_{\text{total}}(f_i)\right]
\]

- 修改MCMC的接受概率，使用动态先验替代原有的静态先验。
- 设计门控网络的训练策略：可采用**元学习**方法，在多个不同领域的数据集上训练门控网络学习最优的专家分配策略。
- 处理门控网络带来的额外计算开销：通过稀疏门控（每次仅激活少数专家）控制成本。

**研究内容四：在双黑洞合并引力波建模中的应用验证**

以双黑洞合并引力波波形建模为应用场景，验证DP-BSR框架的有效性：

1. **问题设定**：从数值相对论模拟数据中自动发现闭合形式的引力波波形模型。
2. **专家先验配置**：
   - 量纲分析专家：确保波形表达式量纲正确（应变是无量纲量）
   - 极限行为专家：确保在合并前（\( t \to -\infty \)）匹配后牛顿近似，合并后（\( t \to +\infty \)）表现为指数衰减的铃宕（ringdown）
   - 参数范围专家：确保质量比 \( q \in (0,1] \)、自旋参数 \( \chi \in [-1,1] \)
   - 无毛定理专家：禁止引入额外自由度
3. **评估指标**：波形吻合度（与数值相对论模拟的匹配程度）、泛化能力（对未见过参数空间的预测精度）、模型可解释性。

## 研究细节

**表达式树的量纲传播：**

每个节点携带量纲标签（如M、L、T）。加法/减法要求子节点量纲相同；乘法/除法允许量纲相乘/相消；初等函数（sin, exp, log等）要求参数无量纲。

**极限行为的自动计算：**

利用符号计算库（Sympy）计算候选表达式在 \( x \to 0 \) 和 \( x \to \infty \) 的渐近行为。

**门控网络的训练：**

采用**离线元学习**策略：先在大量历史数据集上训练门控网络，使其学会根据数据特征（样本量、噪声水平、变量维度等）选择最合适的专家组合。在具体问题中，门控网络可根据当前MCMC采样的模型分布进行**在线微调**。

### 可行性分析

**理论可行性**：本课题建立在Guimerà等人已建立的贝叶斯符号回归理论基础之上。动态先验本质上仍是先验分布，代入贝叶斯公式后不影响后验计算的正确性。

**技术可行性**：BMS的代码已开源。Φ-SO和PhySO已实现了量纲分析约束。MoE在深度学习领域已有成熟实现。各关键技术模块均有可参考的实现。

**数据可行性**：引力波领域有大量公开的数值相对论模拟数据。标准符号回归基准数据集（SRbench）可用于方法验证。
